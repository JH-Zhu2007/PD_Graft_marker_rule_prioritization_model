# Annotated figure guide

This guide annotates the GitHub-facing figure package generated by 12P V4.

Group 06 (`06_optional_context_not_for_strong_claims`) is intentionally excluded from this public-facing figure display package.

## Package groups

- `01_main_single_panel`: core V18 single-panel main evidence figures.
- `02_ml_audit_required_ROC_PR_AUC`: required model-audit figures, including ROC/PR/AUC-related performance evidence.
- `03_publication_panel_package`: publication assembly and panel-package materials.
- `04_supplementary_supporting_evidence`: supporting evidence for maturation, module score, external/context assessment and candidate signatures.
- `05_audit_boundary_reproducibility`: visual audit, caption, reproducibility and no-overclaim/claim-boundary materials.
