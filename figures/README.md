# Figures

This folder contains the GitHub-facing figure package generated by 12P V3 from the 12O final integrated figure package.

The original 12O filenames were shortened for Windows/GitHub path compatibility. The mapping table is:

`manifests/12P_V3_github_public_figure_filename_mapping.csv`

## 12O package groups

- `01_main_single_panel`: 24 PDF files
- `02_ml_audit_required_ROC_PR_AUC`: 4 PDF files
- `03_publication_panel_package`: 145 PDF files
- `04_supplementary_supporting_evidence`: 10 PDF files
- `05_audit_boundary_reproducibility`: 18 PDF files
- `06_optional_context_not_for_strong_claims`: 11 PDF files

## Required ML audit

The ROC/PR/AUC machine-learning audit figures are intentionally retained in `12O_final_integrated_package/02_ml_audit_required_ROC_PR_AUC/`.

## Claim boundary

These figures support candidate transcriptomic prioritisation only. They do not establish clinical prediction, validated biomarkers, anatomical projection proof, barcode-confirmed lineage tracing or genetic causality.
